# Agent Working Rules

> 프로젝트 버전: 0.1.0

이 파일은 이 프로젝트의 플랜, 백엔드, 프론트엔드 작업에서 항상 우선 적용한다.

1. 많은 부분을 수정해야 한다면 반드시 사용자에게 물어보고 진행한다.
2. 하나의 파일에 코드를 다 넣지 말고, 기능별로 모듈화한다.
3. 요청이 명확하지 않을 때 추론 및 실행하지 말고, 우선 사용자의 설명을 제대로 이해했는지 말한다.

## 적용 범위

- Codex가 작성하는 플랜, API 계약, 백엔드 구조, 프론트 구조, 테스트 계획에 적용한다.
- 기능 추가 시 `document_templates`, `investigation_reports`, `generated_text_blocks`처럼 책임 단위로 나눈다.
- HWP/HWPX 파일 자동 작성보다, 사용자가 공문 기록 프로그램에 복사 및 붙여넣기 쉬운 텍스트 블록 생성을 우선한다.
- AI 출력은 최종 행정 판단이 아니라 담당 교사가 검토할 초안으로 취급한다.
